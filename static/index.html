1. Monod Kinetics

μ = μmax · S / (Ks + S)

2. Haldane Inhibition

μ = μmax · S / (Ks + S + S²/Ki)

3. Temperature Effect

μ(T) = μ20 · θ(T-20),  θ ≈ 1.07

4. DO Limitation

μ = μmax · S/(Ks+S) · DO/(KDO+DO)

5. AOB/NOB Model

NH₄⁺ → NO₂⁻ (AOB): dNH₄/dt = -μ·XAOB/YAOB
NO₂⁻ → NO₃⁻ (NOB): dNO₂/dt = μ·XAOB/YAOB - μ·XNOB/YNOB

6. Kinetic Orders

Zero: C = C₀ − kt  |  First: C = C₀e−kt  |  Second: 1/C = 1/C₀ + kt

7. Free Ammonia / FNA

FA = NH₄·10(pH−pKa) / (1 + 10(pH−pKa))
FNA = NO₂ / (1 + 10(pH−pKa))

8. Oxygen Demand

O₂ = 4.57 · ΔNH₄-N  |  Air = O₂ / (0.23 · 0.20)
